Workflow

1. Data Ingestion

Read raw CSV files, split into training and test sets, and store artifacts.

2. Data Transformation

Imputation, scaling, encoding — transforming raw input into ML-ready format.

3. Model Training

Train multiple models, compare, and select the one with the highest R² score.

4. Evaluation

Evaluate performance, log metrics, and save the trained model pipeline.

About the Project

This Student Performance Prediction project is a complete end-to-end ML pipeline, developed with a modular architecture. It offers educators and researchers insights to predict and analyze student outcomes, helping improve academic planning. The system uses best practices in ML deployment, robust preprocessing pipelines, efficient serialization, scalable code structure, logging, and error handling.
